# EW Timing Recovery Lab

## Directions

1. Clone or fork this repository.
2. Run `M&M.py` to see the base BPSK simulation with timing recovery.
3. Work in your assigned group to add **one feature** to the code:
   - **Group 1:** Higher-order modulation (QPSK, 8PSK, 16QAM)  
   - **Group 2:** Bit error rate analysis  
   - **Group 3:** Signal impairments / jamming  
   - **Group 4:** Improved visualization / interactivity
4. Test your feature locally.
5. Push your changes to the shared repo so the class can build a combined simulation.
